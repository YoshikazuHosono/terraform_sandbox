# terraform

## install

```sh
brew tap hashicorp/tap
brew install hashicorp/tap/terraform

# check
terraform -version


# to update
brew upgrade hashicorp/tap/terraform

## tab completion (need to restart shell)
terraform -install-autocomplete
```
